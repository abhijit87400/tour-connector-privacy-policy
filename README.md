**Privacy Policy – Tour Connector**

_Last updated: February 2026_

Tour Connector (“we”, “our”, or “us”) respects your privacy and is committed to protecting the information you share while using our mobile application and WordPress integration services.

This Privacy Policy explains how information is collected, used, and protected when you use the Tour Connector app.

_**1. Information We Collect**_
Tour Connector only collects information necessary for business operations.

_a) Business & Operator Information_

(i) Business name

(ii) Operator name

(iii) Phone number

(iv) Email address

(v) Website endpoint URL

(vi) This information is provided by the user during setup or via QR-based connection.

_b) Customer Lead Information_

When connected to a WordPress website using the Tour Connector plugin, the app may receive:

(i) Customer name

(ii) Phone number

(iii) Email address

(iv) Tour package details

(v) Booking information

(vi) Messages submitted via Elementor forms

_**2. How We Use Information**_

Collected data is used strictly for:

(i) Displaying leads inside the app

(ii) Sending real-time notifications to operators

(iii) Managing bookings, quotations, and invoices

(iv) Generating PDF invoices or quotations

(v) Enabling communication with customers (Call, WhatsApp, Email)

(vi) We do not use data for advertising or marketing purposes.

_**3. QR Code Based Connection**_

(i) Tour Connector uses a QR code scanning mechanism to securely connect the mobile app with a WordPress website.

(ii) No personal data is collected during QR scanning beyond the required API connection details.

_**4. Data Storage & Security**_

(i) Data is transmitted securely between the WordPress plugin and the mobile app

(ii) No unnecessary data is stored on the device

(iii) We take reasonable security measures to protect business and customer information

_**5. Data Sharing**_

We do not sell, trade, or share user or customer data with third parties.

Data is shared only between:

(i) Your WordPress website

(ii) Your connected mobile app

(iii) and only for app functionality.

_**6. Permissions Used**_
The app may request permissions for:

(i) Camera (for QR code scanning)

(ii) Internet access (for syncing data)

(iii) Notifications (for real-time lead alerts)

(iv) These permissions are used strictly for core app functionality.

_**7. Children’s Privacy**_
(i) Tour Connector is a business utility app and is not intended for children under the age of 13.

(ii) We do not knowingly collect data from children.

_**8. User Control**_
(i) Users control which website they connect to

(ii) Users can disconnect the app at any time

(iii) Uninstalling the app removes local access to data

_**9. Changes to This Policy**_
(i) We may update this Privacy Policy from time to time.

(ii) Any changes will be reflected on this page with an updated revision date.
